Dashboard for Introductory Programming Class
  • Description

    Low-fidelity mockups of a dashboard for an introductory programming course offered at UCI. I had to create user interfaces for students, professors, and TA/Readers based on certain criteria given by stakeholders.

  • Skills

    Wireframes using Balsamiq, data visualization

  • Lessons Learned

    I learned how to critically think about how to design a user interface that would be easy to use for different user groups and how to present data in a visual format that is easy to understand.
