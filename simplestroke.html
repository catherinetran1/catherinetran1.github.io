Simple Strokes
  • About

    Simple Strokes is a mobile application that aims to improve fine motor skills for stroke patients. With the choice to either draw or write, patients will be able to trace different letters and shapes to help gain back the skills they had before stroke.

    Click Here to learn more!

  • The Goal

    Create a prototype of a stroke application that can help patients on their road to recovery.

  • Steps to Achieve Goal

    1. Gain domain knowledge about stroke rehabilitation by reading published research articles about stroke and stroke rehab solutions
    2. Conduct gap analysis on current solutions
    3. Brainstorm possible solutions
    4. Create interactive prototype to help visualize solution

  • Skills

    Research, User Centered Design, Adobe XD

  • Lessons Learned

    I learned that gaining domain knowledge is a crucial part when going through the process of developing an innovation in health IT. I learned how to conduct a gap analysis by analyzing published research articles on stroke rehabilitation and reviewing current stroke rehab mobile applications.
